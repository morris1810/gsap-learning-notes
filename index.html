GSAP Animation Note

Basic Method

There are two or more arguments...

1st arrgument : put a string to select class"." or id"#" of the element. UsingquerySeletor() or querySeletorAll()

2nd arrgument : an object to store some animation variable

.to() used for put target from current state animate it to the defined value.

gsap.to("#box1", { x: 150 });

.from() element from defined value to current status

gsap.from("#box2", { x: 150 });

.fromTo() can set the target element from where to where

gsap.fromTo( "#box3", {x : 0}, {x : 300} )

.set() set the property value immediately without animation, zero-duration

gsap.set("#box4", { x: 150 });

Multiple Selection

const box5 = document.getElementById("box5");
const box6 = document.getElementById("box6");
gsap.to([box5, box6], {x: 100});

Animation Variables

Variable Desc.
x: 100 transform: translateX(100px)
y: 100 transform: translateY(100px)
xPercent: 50 transform: translateX(50%)
yPercent: 50 transform: translateY(50%)
rotate: 50 transform: rotate(50deg)
rotateX: 50 transform: rotateX(50deg)
scale: 3 transform: scale(3 3)
scaleX: 3 transform: scaleX(3)
scaleY: 3 transform: scaleY(3)
skewX: 45 transform: skewX(45deg)
skewY: 45 transform: skewY(45deg)
rotateY: 50 transform: rotateY(50deg)
transformOrigin: "0% 100%" transform-origin: 0% 100%;

If using int as the value by default it will be px or deg

If u want to use other units can do calc. in js or put a string for the value

x: 100
x: '100vw'

//Doing calc. and get return value
x: () => window.innerWidth / 2;

Most of the CSS property can use in GSAP just change the hyphen-seperate case to camelCase

It is recommended to use more transfrom and opacity in GSAP, but for the filter and boxShadow are CPU-Intensive

const canvas1 = document.getElementById('canvas1');

Special Properties

Property Desc
duration: 2 second, default=0.5

stagger: 1 if given multiple targets, set a time ( second ) between each target

Timeline

gsap.inline()set a new time line for animation

const tl1 = gsap.timeline();

1.to("#box13", {x: 150, duration: 0.5})
1.to("#box14", {x: 200, duration: 0.5})
1.to("#box15", {x: 100, duration: 1})
//Total 2 second animation for this timeline

add a gap into the timeline by giving the third argument inside tl.to().

<: start of previous animation, may add an integer behind the symbol.

>: end of previous animation, may add an integer behind the symbol.

i: start of timeline

+=i / -=i: end of animation.

value of i can be in percentage(10%) or second(2)

const tl2 = gsap.timeline();

//Add a gap at the begining of animation
tl2.to('#box16', {x: 150}, 1);
//Add a gap with 0.2 second at the end of previous animation (box16)
tl2.to('#box17', {x: 150}, ">0.2");

for visible timeline look here

add some special properties to the timeline can make all the element inside the timeline share the same property value.

const tl3 = gsap.timeline({
repeat: -1,
yoyo: true
});

tl3.to('#box18', {borderRadius: "50%"});
tl3.to('#box19', {borderRadius: "50%"});
tl3.to('#box20', {borderRadius: "50%"});

default may set the default value share to all element in a timeline.

const tl4 = gsap.timeline({defaults: {backgroundColor: "#25f7c9"}});

tl4.to("#box21", {x:150})
.to("#box22", {x:150})
.to("#box23", {x:150});

GSAP Animation Controller

  • play(),pause(),resume(),reverse(),restart().
  • seek(sec): jump to specific second.
  • progress(0~1): jump to the percentage of progress.
    E.g: 0.25 = 1/4
  • timeScale(i): scale the animation time. E.g: 2 = double speed
  • kill(): kill the animation immediately
    • const controlAnimation1 = gsap.to('#box24', {
    x: 200,
    rotate: 360,
    duration: 3
    })
    document.getElementById("play1").addEventListener('click', () => {controlAnimation1.play()});
    document.getElementById("pause1").addEventListener('click', () => {controlAnimation1.pause()});
    document.getElementById("resume1").addEventListener('click', () => {controlAnimation1.resume()});
    document.getElementById("reverse1").addEventListener('click', () => {controlAnimation1.reverse()});
    document.getElementById("restart1").addEventListener('click', () => {controlAnimation1.restart()});
